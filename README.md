# <img src="icon.svg" alt="Pocket Library Logo" style="width:72px"> Pocket Library

Download articles, summaries, analyses, and notes from various English Literature websites as PDF.

## Introduction

Pocket Library uses Paged.js to dynamically convert content from supported websites into well-formatted, ready-to-print documents. Users can directly print or print to pdf using the browser's built in print dialogue.

## Supported Websites
- [CliffsNotes](https://www.cliffsnotes.com/)
- [LitCharts](https://www.litcharts.com/lit) (Literature only)

## Install

Violentmonkey is needed to install the User Script from GitHub. Other similar extensions may also work but have not been tested.

#### (1) Install Violentmonkey for your browser
[![](https://img.shields.io/badge/Google%20Chrome-%231a73e8)](https://chrome.google.com/webstore/detail/violent-monkey/jinjaccalgkegednnccohejagnlnfdag)
[![](https://img.shields.io/badge/Mozilla%20Firefox-%23ff5f3a)](https://addons.mozilla.org/firefox/addon/violentmonkey/)
[![](https://img.shields.io/badge/Microsoft%20Edge-%232cfbcd)](https://microsoftedge.microsoft.com/addons/detail/eeagobfjdenkkddmbclomhiblgggliao)

#### (2) Install the User Script from GitHub
[![](https://img.shields.io/badge/GitHub-User%20Script-%23dc4941)](https://naeembolchhi.github.io/pocket-library/latest)

## Uninstall or Disable

Simply disable or remove the script from the dashboard of your script manager. If you don't understand how that works, uninstall the Violentmonkey extension instead.
